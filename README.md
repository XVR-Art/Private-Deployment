# XVR.Art 私有化部署整合文档

[![XVR元宇宙](https://xvr.oss-cn-hangzhou.aliyuncs.com/common/logo.png)](https://xvr.art)

XVR元宇宙致力于提供虚拟数字空间服务，为数字藏品赋能元宇宙，同时支持教育培训，会议空间，画廊展馆等空间。

私有化部署的客户需自己实现部分API接口, 以定制化自有系统和XVR空间的整合。以下接口均为可选, 按需实现即可, 实现后需联系客服配置前端。

# 更新空间

### 接口地址

{api_gateway}/v1/space/update

#### 业务参数
<table width="100%">
    <tr>
      <th width="25%">参数</th>
      <th>说明</th>
    </tr>
    <tr>
      <td>hub_id</td>
      <td>空间ID, 用于指定要更新的空间</td>
    </tr>
    <tr>
      <td>hub_sid</td>
      <td>空间SID, 唯一, 用于URL, 可以自定义此值来达到与众不同的空间地址, 也可随机修改, 来达到不关闭空间的情况下, 让其他人无法访问<br />空间地址: https://{yourdomain.com}/{hub_sid}</td>
    </tr>
    <tr>
      <td>created_by_account_id</td>
      <td>所属用户, 即用户的 account_id</td>
    </tr>
    <tr>
      <td>scene_sid</td>
      <td>场景SID</td>
    </tr>
    <tr>
      <td>name</td>
      <td>名称</td>
    </tr>
    <tr>
      <td>description</td>
      <td>描述</td>
    </tr>
    <tr>
      <td>room_size</td>
      <td>房间大小, 即可以同时容纳多少人, 范围: 1-50</td>
    </tr>
    <tr>
      <td>entry_mode</td>
      <td>状态, 可选值: allow/deny</td>
    </tr>
</table>

#### 成功返回示例

```json
{
    "code": 0,
    "msg": "OK"
}
```
